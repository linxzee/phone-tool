# Lzee 玩机教程网站

这是一个关于安卓手机刷机、Root、系统修改以及Windows系统重装等玩机教程的静态网站。

**访问地址**: https://phone.dengsir.com

## 📋 教程内容

### 📱 安卓手机教程
- **深刷提取备份刷入（救砖）** - 适用大多数设备，高通设备大多需要拆机找触点
- **Lineage GSI系统的选择** - GSI是通用系统镜像，不看手机品牌型号，都能刷
- **MTKClient 使用教程** - MTK芯片固件的提取、备份与刷入指南
- **Qualcomm 9008 端口备份** - 使用 Qualcomm Premium Tool 在9008模式下备份分区
- **MiKo Pro 9008 操作方法** - 另一款高通9008模式下的固件提取与备份工具
- **红魔手机关闭系统更新** - 通过ADB方式免Root关闭红魔手机的系统更新提示
- **ADB Sideload 使用教程** - 通过电脑向恢复模式设备推送并安装.zip格式更新包的方法。
- **去除WIFI和信号的X或！感叹号** - 解决安卓系统网络连通性检查导致的WIFI/信号感叹号问题。
- **GApps提供商选择指南** - 主流谷歌框架提供商对比与选择指南，适用于各种自定义ROM。

### 💻 Windows系统教程
- **Windows系统重装教程** - Windows电脑通用重装系统详细指南，包含PE工具使用
- **U盘启动快捷键查询列表** - 各品牌电脑和主板的U盘启动快捷键大全

### 🛠️ 实用工具
- **LX Music** - 免费&开源音乐查找工具。

## 🚀 4.0 版本升级亮点 (对比 3.0.2)

### 🎨 CSS 架构重构与优化
- **样式文件合并**: 将 `base.css`, `dark-mode.css`, `docs.css`, `download.css`, `indexstyles.css`, `style.css` 等多个分散的样式文件整合并重构为单一的 [`styles.css`](4.0/main/css/styles.css:1) 文件，极大地简化了样式管理和维护。
- **引入新 CSS 变量**: 增加了更丰富的 CSS 变量，增强了主题定制能力和可维护性。
- **新增专用样式**: 针对下载页面和教程文档新增了专用样式，提升了用户体验和内容展示效果。
- **下拉菜单样式**: 增加了新的下拉菜单组件样式。
- **夜间模式优化**: 对夜间模式的样式进行了全面定义和优化。

### 💡 JavaScript 功能增强与模块化
- **新增通用工具库**: 引入了 [`utils.js`](4.0/main/js/utils.js:1) 文件，封装了主题管理、代码复制、防抖 (debounce)、节流 (throttle) 等通用工具函数，提高了代码的复用性和可维护性。
- **健壮的主题管理**: 主题管理功能更加健壮，支持页面加载时立即应用主题，避免主题闪烁。
- **动态代码复制**: 代码复制功能通过 `MutationObserver` 监听 DOM 变化，支持动态加载的代码块，提升了用户交互性。
- **代码精简**: 移除了 3.0.2 版本中空置的 `indexjs.js` 文件。

### 🔗 导航栏组件统一与路径优化
- **组件统一**: 统一了导航栏组件的命名和引用方式，从 `navbar-root.html` 变为 [`navbar.html`](4.0/main/components/navbar.html:1)。
- **绝对路径引用**: 导航栏组件中的资源路径改为绝对路径，提高了在不同页面层级下的兼容性。

### 📈 整体提升
- **加载性能**: 通过减少 HTTP 请求和优化样式加载，理论上提高了页面加载速度。
- **维护性**: 模块化的代码结构使得后续的功能开发和维护更加便捷。
- **一致性**: 统一的设计语言和代码规范，提升了项目整体的一致性。

## 💻 本地开发

直接在浏览器中打开 `index.html` 即可查看效果，但由于浏览器的安全限制，导航栏的动态加载功能需要本地服务器支持。

推荐使用 VS Code 的 Live Server 扩展：
1. 安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"
